eStyle
========


This module is to create a default plot style environment
for scientific publications.

Features
--------

- Set default plot font.
- Set the number of major and minor ticks.

Installation
------------

Install eStyle by running::

    git clone https://github.com/ejcjason/eStyle.git
    cd eStyle
    python ./install.sh install
